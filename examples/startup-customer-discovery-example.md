# Startup Customer Discovery Example

## Scenario

初期スタートアップが、10件の顧客ヒアリングメモをAIエージェントで整理し、次の検証に使う。

## Input

- 顧客セグメント
- ヒアリングメモ
- 課題仮説
- CRM更新可否

## Expected Output

- 匿名化サマリー
- 課題クラスタ
- 強い証拠、弱い証拠、推測
- 次回ヒアリング質問
- CRM更新案
- 承認待ち操作

## Approval Boundary

AIエージェントは匿名化、分類、下書きまでを担当する。顧客情報の共有、CRM更新、顧客連絡は人間が承認する。
