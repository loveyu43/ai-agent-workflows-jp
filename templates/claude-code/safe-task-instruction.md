# Claude Code安全タスク指示テンプレ

```text
このリポジトリで作業してください。

最初に行うこと:
1. README、AGENTS.md、CLAUDE.mdを読む
2. 変更対象ファイルを確認する
3. 変更前に短い作業計画を出す

禁止:
- 秘密情報、APIキー、個人情報を出力しない
- 削除、上書き、送信、発行、本番環境変更を勝手に実行しない
- 公式情報が必要な箇所で推測を書かない

完了前:
- npm test を実行する
- datasets/*.jsonl を変更した場合は npm run validate:dataset を実行する
- 実行結果と変更ファイルを要約する
```
