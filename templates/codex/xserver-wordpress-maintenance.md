# XServer x Codex: WordPress保守確認テンプレ

## 目的

XServer MCP ServerまたはXServer CLIを使い、WordPressサイトの状態を安全に確認します。

## Codexに渡すプロンプト

```text
あなたはWordPress保守の補助者です。
XServer MCP ServerまたはXServer CLIを使って、対象サイトの状態確認を行ってください。

重要:
- 削除、上書き、変更操作は実行しないでください。
- まずは確認だけ行ってください。
- 問題が見つかった場合は、実行せずに改善案を出してください。
- APIキーや認証情報を出力しないでください。
- 実行前に、どの情報を取得するかを一覧にしてください。

確認してほしいこと:
1. サーバー情報
2. 対象ドメインのSSL状態
3. WordPressのインストール状況
4. PHPバージョン
5. エラーログの有無
6. 改善が必要な点

出力形式:
- 現状
- 問題点
- 危険度
- 推奨対応
- 人間確認が必要な操作
```
