# AIエージェント安全利用の基本

この文書はセキュリティ監査ではありません。非エンジニアや小規模チームがAIエージェントに業務を任せる前の基本チェックリストです。

## AIに任せやすい作業

- 情報収集
- 状態確認
- 手順書の作成
- エラー原因の候補出し
- 文章案、投稿案、チェックリスト案の作成
- 差分レビュー

## 人間確認が必要な作業

- DNS変更
- SSL設定変更
- WordPress削除
- MySQL削除
- 請求書発行
- 会計データ変更
- 顧客情報の取得や出力
- メールアカウント作成、削除
- 本番環境への反映

## 指示文に必ず入れる一文

```text
削除、上書き、送信、発行、課金、個人情報の出力、本番環境の設定変更は、事前に実行内容を説明し、人間の明示承認を得るまで実行しないでください。
```

## 最初の運用ルール

- AIにはまず「確認だけ」を依頼する
- 実行前に「何を変えるか」を箇条書きにさせる
- 高リスク操作は別チャットではなくIssueやPRで記録する
- 失敗したらロールバック手順を先に確認する
