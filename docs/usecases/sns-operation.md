# SNS運用AI補助ユースケース

## 目的

SNS投稿案の作成、事実確認、炎上リスク確認、投稿カレンダー作成をAIエージェントで補助します。

## AIに任せる範囲

- 投稿案の作成
- 表現のトーン調整
- 事実確認チェックリスト作成
- 誇大表現の検出
- 投稿カレンダー案

## 人間確認が必要な範囲

- 実際の投稿
- キャンペーン価格や条件
- 医療、法律、金融に関わる表現
- 他社や顧客への言及
- 画像や引用の権利確認

## 安全な指示例

```text
投稿案を10本作ってください。
各投稿に、事実確認が必要な箇所、炎上リスク、誇大表現の有無、人間確認ポイントを付けてください。
自動投稿はしないでください。
```
