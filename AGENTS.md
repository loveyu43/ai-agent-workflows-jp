# Agent Instructions

Use this repository as a documentation and dataset project.

## Rules For AI Agents

- Prefer official sources and include links when adding provider notes.
- Do not invent current product capabilities.
- Do not add secrets, real customer data, private logs, or credentials.
- Mark destructive operations as human approval required.
- Run `npm test` before claiming changes are ready.
- Run `npm run validate:dataset` after editing `datasets/*.jsonl`.

## Style

- Write for Japanese non-engineers first.
- Keep templates copy-paste friendly.
- Use cautious language for security topics.
- Separate "confirm only" actions from "change production" actions.
