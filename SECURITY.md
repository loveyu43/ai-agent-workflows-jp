# Security Policy

This repository contains templates and educational material. It does not provide a security audit, legal advice, or production approval.

## Scope

Please report issues when a template could cause unsafe behavior, such as:

- Exposing API keys or secrets
- Encouraging destructive operations without human approval
- Mixing production data into examples
- Missing warnings for DNS, SSL, database, billing, accounting, or personal data changes
- Suggesting that AI agents should run irreversible operations automatically

## Reporting

Open a GitHub issue with:

- Affected file path
- Why the wording or workflow is risky
- Safer wording or behavior if you have a suggestion

Do not include real credentials, customer data, invoices, logs, or private system details.
