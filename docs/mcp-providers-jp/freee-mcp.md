# freee-mcp

Last checked: 2026-06-02 JST

Official URL: https://corp.freee.co.jp/news/20260302freee_mcp.html

GitHub: https://github.com/freee/freee-mcp

## これは何か

freee-mcpは、AIエージェントからfreeeのAPIを操作するためのMCPサーバーです。freeeの公式発表では、会計、人事労務、請求書、工数管理、販売などのAPIをMCPツール化し、npmパッケージとして公開されていると説明されています。

## できること

- freee APIをAIエージェントから扱う
- 会計、請求書、人事労務、工数管理、販売などの業務を補助する
- Claude Desktop、Claude Code、Claude Cowork、Cursorなどから利用する

## 向いている人

- freeeを使っている小規模事業者
- バックオフィス業務をAIで補助したい人
- 請求、会計、人事労務の手順を標準化したいチーム

## 危険度が高い操作

- 請求書発行
- 取引先登録や変更
- 会計データ変更
- 個人情報や給与関連情報の取得
- 認証情報の共有

## 安全な使い方

請求書や会計データは業務影響が大きいため、AIには下書き、差分確認、入力候補の整理までを任せます。発行、登録、変更、送信は人間承認を必須にします。

## AIに渡す指示例

```text
あなたはfreee業務の補助者です。
請求書や会計データを扱う場合、まず下書きと確認項目だけを作成してください。

以下は人間の明示承認があるまで実行しないでください。
- 請求書の発行
- 取引先の登録や変更
- 会計データの変更
- 個人情報、給与情報、顧客情報の出力
- 外部への送信

出力には、確認すべき取引先、金額、日付、税区分、送信先、承認者を含めてください。
```
